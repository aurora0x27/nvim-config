## 功能

- 仅包含基本但必要的功能

- 代码结构清晰，便于初学者理解

## ⚡立即体验⚡

无需替换原始配置即可立即体验

```bash

./venv # 启动一个虚拟环境 shell，`nvim-debug` 将自动添加到路径

vi # 使用此配置启动 Neovim，无需更改 ~/.local/share

```

## 基本功能

- [x] 自动补全

- [x] 状态栏

- [x] 颜色和注释高亮

- [x] 文件系统资源管理器

- [x] Markdown 预览

- [x] Markdown 渲染器

- [x] 大纲

- [x] 集成终端

- [x] 模糊查找器

- [x] LSP 支持

- [x] 格式化程序

- [x] SSH 剪贴板支持（**需要 tmux 额外配置**）

- [x] 输入法自动切换（**仅限 Linux 和 macOS**）

- [x] 工作区补丁

- [x] Typst 支持

- [x] 任务运行基础设施 (**Overseer**)
